MSB Extractor bookmarklet

Installs a one-click capture button that scrapes your logged-in MyStrengthBook session and downloads a JSON file you can feed into the msb-extractor Python CLI.

Install

  1. Show your browser's bookmarks bar if it's hidden.
  2. Drag the blue button below onto the bookmarks bar.
  3. Open app.mystrengthbook.com and sign in.
  4. Click the bookmark. Watch the console for progress; a file named msb_capture.json will download when it's done.
